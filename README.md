

# MCDR-API-PluginID

> ——让插件拥有一个独一无二的ID



## 它有什么用?

```
1. 让开发者不再担心插件重命名后无法使用
2. 让腐竹随便重命名插件都行
```



## 开发者该如何使用它来提(降)升(低)开发效率呢?

我们提供了一个[开发文档](./docs/development.md)和一个[示例插件](./example_plugin.py)



# 注意! 注意!! 注意!!!

尽量不要把[MCDR-API-PluginID](https://github.com/SFR-Server/MCDR-API-PluginID)作为插件的必要前置，这会导致腐竹们必须多下载一个插件:)
